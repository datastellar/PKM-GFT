# G6 — MDDS for Domains: Standardisasi Atribut Data E-Government

## 📖 Kutipan Lengkap (Format Sementara)
> National e-Governance Division (NeGD) India. (2022). *Metadata and Data Standards (MDDS) for e-Governance Domains Documentation*. Government of India. https://egovstandards.gov.in/metadata-and-data-standards-mdds-domains-documentation

---

## 📌 Ringkasan
Dokumen teknis eksekusi dan *source code level* dari arsitektur e-Gov India. Jika G1 membahas *konsep* domainnya, G6 menangani penetapan **baku mutu atribut (template standar)** tiap domain tersebut.

Contoh sederhana: "Alamat". Sebuah kementerian mencetak "Alamat" menjadi 1 baris string. Kementerian lain mencetak "Alamat" menjadi 5 field (Jalan, RT, RW, Kodepos, Kota). Perbedaan format receh ini menggagalkan milyaran interaksi API publik. G6 ini merekam solusi memaksakan **Satu Standar Metadata Nasional** (Satu cara tunggal merekam kependudukan) untuk seluruh ekosistem di benua tersebut.
